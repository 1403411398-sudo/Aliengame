# Aliengame
本项目基于 Alien Invasion 项目进行开发，是一个经典的 2D 飞船射击游戏。并在原基础上进行了多项扩展和功能优化，增加了护盾和外星人攻击功能
# 🚀 项目特色（本次实验新增功能）
# 1. 外星人主动攻击功能（新增）
新增了 AlienBullet 类，每个外星人具有主动攻击能力：  
外星人会定时向下发射子弹  
子弹有独立速度、独立绘制  
玩家飞船需要躲避外星子弹  
提高了游戏难度和可玩性  
# 3. 玩家护盾系统（新增）
为玩家飞船添加了 可破坏护盾：  
护盾有生命值  
可被玩家子弹 & 外星人子弹击毁  
护盾破坏后可以移除  
增强游戏策略性  
# 安装与运行
#克隆项目  
git clone <repository-url>  
#进入项目目录  
cd alien-invasion  
#安装依赖  
pip install pygame  
#运行游戏  
python main.py
